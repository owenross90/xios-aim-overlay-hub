## What it Does

Xios Aimbot is a compact, browser-hosted utility built to support aiming in first-person shooter games that run in the browser. It places a responsive crosshair overlay on top of the game view and shifts it using visual cues from the scene. Because it operates entirely inside the browser, there is no installation step and no system-level changes, which makes it easy to use on different machines when needed.

This release centers on faster target acquisition with adjustable crosshair appearance and sensitivity controls. Updated for 2026, it improves overlay rendering for smoother operation and extends compatibility to more browser games. The script is meant as a practice aid for players who want to sharpen reaction time and build more reliable muscle memory in high-tempo shooting situations.

## Common Questions

**How do I install the script?**  
There is nothing to install. Download the HTML file and open it in your browser. It runs locally from the file itself.

**Will my settings survive updates?**  
Yes. Your preferences are stored in the browser's local storage and remain available after updates. Clearing cache data can reset them.

**Can I adjust the crosshair more?**  
Yes. The options panel lets you change color, size, opacity, and smoothing. If you want deeper customization, you can edit the script source directly.

**Does it work on every website?**  
No. It is intended for browser games using canvas-based rendering. Sites that block overlays or rely on unusual rendering methods may not work correctly.

**Where is my data kept?**  
Everything is saved locally in your browser. The script does not send data to external servers or collect user information.
